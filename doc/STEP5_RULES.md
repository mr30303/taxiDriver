# Step 5 Rules Summary (급여 계산 규칙 요약)

이 문서는 Step 5 구현 전에 확정된 규칙을 정리한 것입니다. (구현 없음)

## 1) 입력 구분
### 초기 설정값
- 기준금(사납금) (기본값: 4,550,000)
- 성과급 비율 (기본값: 0.6)
- 기본급
- 만근 기준 일수 (기본값: 26)
- 만근 기준 월 승무수당
- 상여금
- 무사고 수당
- 가맹 수당
- 연차 인정금(1일) (기본값: 115,000)
- 법정공휴일 인정금(1일) (기본값: 87,000)

### 일일 입력값
- 일 실입금(운송수입)
- 톨게이트비 (월 합계 표시용, 급여 계산 제외)
- 근무 형태 (일반 / 연차 / 법정공휴일)
- 사고 여부

## 2) 월 단위 자동 계산값
- 월 실입금 합계
- 월 톨게이트비 합계
- 연차 일수 / 법정공휴일 일수
- 인정금 합계
- 만근 여부
- 성과수당
- 승무수당
- 세전 월급

## 3) 계산 규칙
### 매출/인정금
- 월 실입금 합계 = 일 실입금 합
- 인정금 합계 = 연차일수 * 연차 인정금(1일) + 법정공휴일일수 * 법정공휴일 인정금(1일)

### 만근 판단
- (월 실입금 합계 + 인정금 합계) >= 기준금 -> 만근
- 만근 여부는 근무일수와 무관

### 성과수당
- 초과운송수입 = (월 실입금 합계 + 인정금 합계) - 기준금
- 초과운송수입 <= 0 이면 성과수당 = 0
- 성과수당 = 초과운송수입 * 성과급 비율

### 승무수당
- 일 승무수당 = (만근 기준 월 승무수당 / 만근 기준 일수)
- 일 승무수당은 **백원 단위 반올림**
- 승무 인정 일수 =
  min(만근 기준 일수, 만근 기준 일수 * (월 실입금 합계 + 인정금 합계) / 기준금)
- 승무수당 = (반올림된 일 승무수당) * (승무 인정 일수)

### 고정 수당
- 기본급: 항상 포함
- 상여금: 만근 시 지급
- 가맹 수당: 만근 시 지급
- 무사고 수당: 만근 + 사고 없음일 때 지급

### 총액(세전 월급)
- 세전 월급 = 기본급 + 성과수당 + 승무수당 + (만근 조건 수당들)
- **총액은 백원 단위 반올림**

## 4) 결과 화면 표시(최종 출력)
- 월 실입금 합계
- 월 톨게이트비 합계
- 세전 월급

## 5) 비고
- 톨게이트비는 급여 계산에서 제외 (월 합계만 표시)
- 회사 내부 정산 로직을 그대로 재현하지 않음
