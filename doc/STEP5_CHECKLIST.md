# Step 5 Verification Checklist (급여 계산기 검증)

## 1) 빌드/설치
- [x] `.\gradlew assembleDebug` 성공
- [x] `.\gradlew installDebug` 성공 (에뮬레이터 연결)

## 2) 설정 화면 (SalarySetting)
- [ ] 기본값 표시 확인 (기준금 4,550,000 / 성과급 0.6 / 만근일수 26)
- [ ] 값 변경 후 저장 시 유지됨

## 3) 일일 입력 화면 (DailySalesInput)
- [ ] 날짜/일 실입금/톨게이트비 입력 가능
- [ ] 근무 형태 선택 가능 (일반/연차/법정공휴일)
- [ ] 사고 여부 체크 가능
- [ ] 추가 버튼으로 리스트에 반영됨
- [ ] 삭제 버튼 동작

## 4) 계산 규칙 확인 (핵심)
- [ ] 월 실입금 합계 = 일 실입금 합
- [ ] 월 톨게이트비 합계 = 톨게이트비 합 (급여 계산에는 미반영)
- [ ] 인정금 합계 = 연차일수*115,000 + 공휴일일수*87,000
- [ ] 만근 판단: (월 실입금 + 인정금) >= 기준금
- [ ] 성과수당 = max(0, 초과운송수입) * 성과급 비율
- [ ] 승무 인정 일수 =
      min(만근 기준 일수, 만근 기준 일수 * (월 실입금+인정금) / 기준금)
- [ ] 일 승무수당 = (만근 기준 월 승무수당 / 만근 기준 일수) → 백원 반올림
- [ ] 총액(세전 월급) = 기본급 + 성과수당 + 승무수당 + (만근 조건 수당)
- [ ] 총액 백원 단위 반올림 적용

## 5) 결과 화면 (SalaryResult)
- [ ] 월 실입금 합계 표시
- [ ] 월 톨게이트비 합계 표시
- [ ] 세전 월급 표시

## 6) Firestore 연동 (선택 확인)
- [ ] 설정 저장 시 Firestore에 저장됨
- [ ] 일일 입력 저장 시 Firestore에 저장됨
